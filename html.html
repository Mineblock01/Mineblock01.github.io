Actividades de html

Volver

Durante estas últimas semanas hemos ido haciendo trabajos de html. Estos trabajos consistian en intentar replicar un documento que nos daban. Los elementos html con los que más trabajabamos eran <p>, <a gt, <br>, <h1> y otras. Ahora veremos que hace cada una.
<p>: Se utiliza para marcar cuando un elemento forma un parrafo.
<a>: Indica un hiperenlace, o link. Para marcar la dirección, que puede ser absoluta o relativa, se utiliza el atributo href="aquí va la dirección".
<h1>, <h2>, etc: Estos elementos se utilizan para nombrar titulos, subtitilos y otros encabezados. El número indica la importancia deste.
<title>: Se utiliza cuando queremos cambiar el titulo de las pagina.
<hr>: Es una linea en la página que la divide en dos.
<br>: Es un salto de linea.
<b>: El texto se resalta en negrita.
<i>: El texto se resalta en curviva.
<link>: Se utiliza para establezer una relación entre el documento y otro recurso, por ejemplo para añadir un css, establece una versión alternativa, el favicon u otros.
<img>: Indica una images, se le puede indicar el tamaño y un texto alternativo entre otras.
<sript>: Hace referencia a un codigo de javascript que corren en el navegador usuario.

Elementos más complejos

Listas ordenadas y desordenadas:

Se indican on <ol> y <ul> respectivamente. Ambas utilizan el elemento <il> para marcar cada entrada de la lista. Por lo general, a menos que el css o js lo cambie, las listas no aportan ningún tipo de diferencia visual en comparación con otros elementos, pero está bien añadirlas ya que le aportan más significado al html.

Tablas

Las tablas son más complejas. Se definen con <table>. Para crear una nueva fila, se utiliza <tr>, mientras que <tdd>. Para la primera linea tambien se puede utilizar <th>. Solo las celdas (y una celda <th> de encabezado) pueden ser elementos secundarios de una fila . Y la fila <tr> en sí misma no puede ser un elemento secundario de una celda. Esto limita la posible anidación de etiquetas. Se pueden utilizar los atributos colspan y rowspan para fusionar celdas.

Formularios

Por último, los formularios html se definen con <form>, el atributo action induca la url que ha de procesar el formulario. Los elementos más basicos de un formulario son <label>, que indica una entrada, <input> que es a entrada de datos. También se puede utilizar &tltextrea> que define un area de texto, <select> que define una selección, con los distintos elementos a selectionar indicados después con <option>. Por último, se pueden añadir botones.
