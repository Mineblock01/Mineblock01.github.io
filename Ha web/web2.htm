La Web 2.0

Indice: Redes Sociales RSS

La web 2.0, o web social, es la web en la que estamos viviendo. Apareció en los mediados de los 2000 y esta caracterizada por su dinamismo y capacidad de interacción. Con la web 2.0 desapareció la diversidad de protocolos y metodos de comunicación, y se intrauro el monopolio del navegador que utilizaba HTTP, y porteriormente HTTPS. Foros que antes se accedian en IRC ahora lo hacen en paginas web, y el correo electronico, ahora se accede mediante el navegador. Sin embargo, para contrarrestar esta unidad, la web desarroyaria nuevas tecnologías, más notablemente Ajax(1999), RSS(1999), JSON(2006), WebRTC(2011) y Wasm(2017), que le daran más potencia al navegador, a costa de hacerlo más costoso de desarollar, causando que se use casi exclusivamente Google Chrome como medio para navegar la red. Otro efecto de la web 2.0 es que, con nuevas librerias de javascript, estan apareciendo programas que en realidad son paginas web. Con la web social aparecen nuevos servicios, los Blogs, las Wikis, las Redes Sociales (RRSS).

Redes Sociales

Las redes sociales (o RRSS) son servicios que permiten la comunicación entre individuos mediante el internet. Se diferencian de otros metodos de comunicación, e-mail e IRC, ya que se ha de crear un usuario con caracter público o semi-público con el que se puede establecer comunicación con otros usuarios de la red. Las redes sociales más utilizadas y conocidas son Intagram y Facebook (Meta), Twitter (X Inc), Youtube (Google) y TikTok (ByteDance). Todas estás tienen unas caracteristicas centrales comunes. En ellas el usuario pública algun tipo de contenido, una imagen, un video o un poco de texto, y otros usuarios consumen este contenido, que normalmente se le presenta de forma algoritmica, y responden a él, comentando y respondiento para los más activos, o dando un me gusta.

RSS

RSS (Sindicación realmente sencilla, del inglés Really Simple Syndication) es un formato para distribuir información en la web. RSS es formato de XML. Se usa para paginas que necesitan distribuir información actualizada, como periodicos y agregadores de noticias. RSS tambien se usa con otros fines, estos pueden podcasts, para que el usuario siempre tenga el último episodio disponible, tambien hay redes sociales que dan la opción de seguir a usuarios mediante rss, blogs tambien lo usan, y en conjunción a Bittorent, puede usarse para automaticamente descargar el contenido una vez publicado. RSS es similar a otras tecnologías, como Atom, y subscripciones de correo o "Newsletters".
